
### Cách chạy project
- Cài đặt backend: 
``` git clone https://github.com/Nguyenthithuhang0406/QuanLyKhoHang_BE.git```
- Lệnh chạy backend: 
```
npm i
npm run dev
```

- Cài đặt frontend
```
git clone https://github.com/Nguyenthithuhang0406/QuanLyKhoHang_FE_TTCSN.git
```
- Chạy frontend
```
npm i
npm run dev
```
